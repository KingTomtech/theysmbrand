# The YSM Brand

Static website for The YSM Brand, a Lusaka-based wellness brand focused on skincare, culinary experiences, meal prep, and fitness nutrition.

## Site structure

- `ysm-brand/index.html` - homepage and brand overview
- `ysm-brand/shop.html` - skincare products and bundle inquiries
- `ysm-brand/meal-prep.html` - meal prep request flow
- `ysm-brand/brunch.html` - private brunch and culinary booking inquiries
- `ysm-brand/fitness.html` - fitness meal-plan positioning
- `ysm-brand/about.html` - brand story and social proof
- `ysm-brand/contact.html` - WhatsApp, Facebook, and Instagram contact paths

The site is configured for Cloudflare Workers/Pages static assets through `wrangler.jsonc`.
